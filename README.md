<a href='https://bmcbioinformatics.biomedcentral.com/articles/10.1186/s12859-018-2509-3'>2剤併用の効果を予測する</a>: NCI-ALMANACリソースデータを活用。DragonというQSAR予測の際によく使われる商用ベースのケモインフォマティクスソフト (2019年6月で販売終了) を使って大量のchemical descriptorをまとめた行列を用意。2剤の行列をconcatenateして学習させたもののと、その細胞株におけるmRNA発現、miRNA発現、タンパク発現情報をそれぞれconcatenateして、FCでは残差接続をしてIC50を出した。実際には遺伝子発現だけでもかなりの精度になっている。<a href='https://github.com/ECP-CANDLE/Benchmarks/tree/release_01/Pilot1/Combo'>Combo</a>のコードはこちら。臨床データはない。<br>
<a href='https://www.nature.com/articles/s41467-017-01153-8'>遺伝子isoformが抗がん剤感受性を決めている</a>: 公共データをいろいろ使ってドライだけでNature Communications 2017に掲載している。データはそれほどすごくない。細胞株への抗がん剤感受性データをvalidationしているが、これもドライ。何かしらの未来を見せないといけない。<br>
